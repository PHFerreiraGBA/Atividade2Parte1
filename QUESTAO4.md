# Questão 4

## As vantagens de usar Template Strings

Template Strings são uma nova forma de formatação de strings adicionada no EcmaScript 6. Vocês podem estar se perguntando:
"Mas isso já dá para fazer no JS por meio de concatenação! Porque usar essa nova forma se a concatenação funciona
perfeitamente, hein"? Bem, há várias vantagens adicionadas no uso de Template Strings. Dessas, vou dizer três. 

Primeiro: a facilidade de leitura. Template Strings são, de uma forma hiperbolizada, mil vezes mais fáceis de ler do que
strings concatenadas. É mais fácil de ler e escrever e é mais limpo, menos informação nos seus olhos, mais calma para o
programador. 

Segundo: a criação de strings multilinha sem precisar de quebrar linha. Nas Template Strings, você não precisa dar /n 
toda vez que quiser quebrar uma linha na string. Basta dar um enter que ele não vai dar erro, ao contrário das strings 
comuns, que necessitam de um /n toda vez que esse caso se torna verdade.

Terceiro e último da lista: a interpolação de expressões. Com as Template Strings, você pode facilmente inserir o
resultado de uma expressão sem precisar criar uma nova variável. Basta colocar a expressão entre chaves (`${assim}`).
Pela concatenação, mesmo para resultados de expressões simples e pequenas, você precisava criar uma nova variável
e juntar ela com as outras partes da string.

E tem muito mais vantagens do que essa, mas esse já está um resumo bem atrativo do que as Template Strings podem nos
oferecer. Ouse testar essa nova opção. Vai valer a pena.
