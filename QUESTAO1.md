# Questão 1

## O que é JSON e o que o popularizou?

JSON (JavaScript Object Notation) é um formato textual que serve para o intercâmbio de dados entre
sistemas computacionais. Antes do JSON, os programadores usavam o XML para essa função. Porém, a
simplicidade na escrita do JSON e facilidade de uso tornou-o popular entre os programadores.
Além disso, dados escritos em JSON são mais fáceis de serem analizados e gerados pela máquina
do que dados XML. Ao menos é o que diz o site oficial do JSON ([json.org](https://www.json.org/json-pt.html)).

Seus padrões estão definidos no padrão ECMA-404. No momento da escrita desse texto, o padrão já 
está em sua 2ª edição. Em sua forma de escrita, ele é independente de linguagem, o que torna ele
um ótimo formato para troca de informações. Essa é mais outra característica que ajudou a
popularizar o seu uso.
